# Pan Pa Ya — App Móvil (Flutter)

Maqueta de la app para conductores y auxiliares en calle.

## Pantallas

1. **Selección de placa** — lista desplegable con vehículos del día
2. **Hoja de ruta** — clientes con estados, notas, botones Novedad / Tomar Foto

## Ejecutar

```bash
cd mobile
flutter pub get
flutter run
```

## Próximo paso

Conectar a Supabase (`supabase_flutter`) para placas y pedidos en tiempo real.
